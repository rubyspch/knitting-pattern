Knitting Pattern Creator

How to use:

This web page is intended to be used to create a grid for a knitting pattern. You can find a list of the supported colour names here.

  1. Enter your main pattern colour in the space and click 'Change background'
  2. Enter your secondary colour and click 'Change colour'
  3. Now you can click the squares you wish to change to your secondary colour to create your pattern
  4. You can change the colour as many times as you need to create your pattern
