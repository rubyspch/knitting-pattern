Knitting Pattern Creator

How to use:

This web page is intended to be used to create a grid for a knitting pattern.

  1. Add the dimensions of the required pattern square, stitches x rows
  2. Select your main pattern colour above 'Main colour' then click the button to set it
  3. Enter your secondary colour and click the button to set it
  4. Now you can click the squares you wish to change to your secondary colour to create your pattern
  5. You can change the colour as many times as you need to create your pattern
